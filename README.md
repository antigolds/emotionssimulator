## Features

- **Emotional sandbox:** Paint directly onto the canvas using feelings that behave like physical matter (liquids, gases, solids).
- **Dynamic alchemy:** Observe complex interactions as elements mix, conduct heat, or explode based on their proximity.
- **Lightweight:** A single page containing all the HTML, CSS, and JS code.
